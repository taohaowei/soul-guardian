# Soul Guardian 产品定位

## 一句话定位
- **中文**：一位住在终端里的心理咨询师，越聊越懂你。
- **英文**：A therapist that lives in your terminal — and remembers every conversation.

## GitHub tagline
> A persistent AI counselor for Claude Code — it remembers you, speaks Chinese, and keeps everything local.

## 品牌调性

| 维度 | 定义 | 表达边界 |
|------|------|---------|
| 温度 | 温暖而不煽情 | 像一个靠谱的老友，不是鸡汤公众号 |
| 专业 | 有方法论但不秀术语 | 提到专业概念时必须"翻译"成人话 |
| 诚实 | 坦承 AI 的局限 | 永远不暗示自己能替代真人咨询师 |
| 文化 | 中国家庭语境原生 | 婆媳、鸡娃、原生家庭——说中国人听得懂的话 |
| 安全 | 隐私第一 | 反复强调"数据全在你自己电脑上" |

## 核心价值主张
1. **它记得你** — 持久记忆系统，不是每次都失忆的 GPT 对话
2. **数据是你的** — 所有档案存在本地，没有服务器、没有上传
3. **懂中国家庭** — 婆媳、亲子、原生家庭、代际传递
4. **不像填问卷** — 渐进式建档，4 次对话自然积累
5. **危机有兜底** — 内置风险识别和中国心理援助热线

## 差异化卖点排序
1. 持久记忆 + 渐进建档（vs 所有竞品的单次对话）
2. 中文原生 + 中国家庭场景（零竞品）
3. 数据全本地（vs 云端存储的工具）
4. 多流派融合（vs Therapy Mode 偏 CBT）
5. 危机安全协议

## 目标场景
- 深夜加班结束，觉得"今天好累、好没意思"
- 和婆婆/父母大吵一架后，想找人说说
- 对孩子发火后的内疚
- 周末做完满意度评估，想聊聊
- 连续失眠，知道是焦虑但不知道怎么处理
